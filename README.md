## Getting Started

Follow these steps to run the project locally:

1. Clone the repository:
   ```
   git clone
   ```

2. Navigate to the application directory:
   ```
   cd countries_app
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Start the application:
   ```
   npm start
   ```

5. In your web browser navigate to:
   ```
   http://localhost:3000
   ```
